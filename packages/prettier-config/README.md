# @hooni-labs/prettier-config

Shared Prettier configuration for Hooni Labs projects.

## Usage

In `.prettierrc`:

```json
"@hooni-labs/prettier-config"
```

Or in `package.json`:

```json
{
  "prettier": "@hooni-labs/prettier-config"
}
```

## Features

- Tailwind CSS plugin
- Consistent formatting
- 100 character line width
- Single quotes
- No semicolons
