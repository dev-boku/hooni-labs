# @hooni-labs/eslint-config

Shared ESLint configuration for Hooni Labs projects.

## Usage

```json
{
  "extends": "@hooni-labs/eslint-config"
}
```

## Features

- TypeScript support
- React support
- React Hooks support
- Import sorting
- Custom rules for consistency

## Rules

- No unused variables
- No explicit any (warn)
- React best practices
- TypeScript strict checking
